#  Sistema de Boletim Escolar

Projeto desenvolvido em **HTML + JavaScript** com o objetivo de simular um boletim escolar, aplicando conceitos de lógica de programação.

---

##  Funcionalidades

* Gerar boletim ao clicar em um botão
* Calcular média dos alunos
* Classificar alunos:

  * Excelente
  * Aprovado
  * Reprovado
* Exibir resultados na tela

---

##  Regras de Classificação

* Média **≥ 9** → Excelente
* Média **≥ 7** → Aprovado
* Média **< 7** → Reprovado

---

##  Conceitos utilizados

* Estrutura de repetição **for**
* Estrutura de repetição **while**
* **Vetores (arrays)**
* **Matrizes (arrays dentro de arrays)**
* Manipulação de HTML com JavaScript
* Boas práticas de **Clean Code**

---

##  Interface

* Título da página
* Botão "Gerar Boletim"
* Área de exibição dos resultados

---

##  Exemplo de saída

Aluno 1: Média 7.0 → Aprovado
Aluno 2: Média 5.0 → Reprovado
Aluno 3: Média 9.0 → Excelente

---

##  Extras implementados

* Total de alunos aprovados
* Maior média
* Menor média
* Interface estilizada com CSS
